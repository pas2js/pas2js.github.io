Example code : Using Copy to create a new dynamic array in Pas2JS

// Defining an array

var

  a, b : array of Integer;

 

begin

// Instantiate an array

  a := new Integer[5];

  b := a.Copy();

  a[1] := 10;

  WriteLn( b[1] );  // 10

 

 

  a.SetLength(10);

  a[1] := 20;

  WriteLn( b[1] );  // 10

Result is: 10 and 10
