TrimLeft
Function		 Removes leading blanks from a string SysUtils unit

Description
The TrimLeft function removes blank and control characters (such as line feed) from the start of a string.
Related commands
Delete Delete a section of characters from a string

TrimRight Removes trailing blanks from a string
 
Example code : Illustrating Trim, TrimLeft and TrimRight
const
  PaddedString = '  Letters     ';
begin
  WriteLn('[' + TrimLeft(PaddedString)  + ']');
  WriteLn('[' + TrimRight(PaddedString) + ']');
  WriteLn('[' + Trim(PaddedString)      + ']');
end;
